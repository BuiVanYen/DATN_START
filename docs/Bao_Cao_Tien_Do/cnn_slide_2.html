CÁCH THỨC TÍCH HỢP AI VÀO HỆ THỐNG

Machine Learning Pipeline

Quy trình 4 bước chuyển mô hình AI cứng nhắc thành Dịch vụ API thời gian thực

1

Huấn Luyện (Môi trường ngoài)

Sử dụng Google Colab (với GPU T4) đắm mình vào hàng ngàn ảnh lá xà lách. Huấn luyện ra mô hình keras cân định sẵn hệ số.
model.fit(train_dataset)
model.save('leaf_disease.h5')
2

Tối ưu & Ép Kiểu (TFLite)

File .h5 thường rất năng (50MB+), ép nó về ranh giới TensorFlow Lite (.tflite) chỉ còn vài MB, nhẹ tâng khi nhúng vào Raspberry Pi.
converter = tf.lite.TFLiteConverter...
tflite_model = converter.convert()
3

Hosting bằng FastAPI

Đưa file .tflite chạy thường trực thông qua tflite-runtime. FastAPI khai báo Endpoint đón dữ liệu ảnh để gọi mô hình phân tích.
@app.post("/api/disease")
async def detect(file: UploadFile):
  return inference(file)
4

Tương tác End-User (Web UI)

Trình duyệt điện thoại kích hoạt Camera HTML5, user chỉ bóp cò. Ảnh gửi dạng form-data lên server và hứng về JSON %.
<input type="file"
       capture="camera">
→ { "disease": "Thối rễ" }
