VAI TRÒ CỦA FASTAPI TẠI BACKEND

Trái tim hệ thống

Kiến hình mạng nhện trực quan cho "bộ não" điều phối tín hiệu và dữ liệu

Hiệu Năng Tốc Độ Cao

Nhanh ngang ngửa NodeJS và Go nhờ hỗ trợ lập trình bất đồng bộ (async/await), giúp không bị nghẽn khi hàng loạt cảm biến gửi dữ liệu về cùng lúc.

Kiểm Soát Chặt Chẽ

Sử dụng Pydantic để bắt rào lỗi dữ liệu đầu vào. MQTT hay Web gửi sai định dạng (ví dụ: nhiệt độ là chữ thay vì số) sẽ bị lọc và từ chối liền.

Tự Tạo Tài Liệu API

Tự sản sinh ra giao diện Swagger UI kiểm thử API chuyên nghiệp tự động, giúp việc ghép nối web/thiết bị dễ dàng hơn.

5 Trụ Cột Điều Phối

FASTAPI

Main Backend Core
Giao Tiếp Thiết Bị

Lắng nghe, bóc tách chuỗi JSON từ MQTT Broker gởi về, đồng thời gửi lệnh PWM khiển quạt, máy bơm.

Lưu Trữ Lịch Sử

Chuyển đổi dữ liệu thô sang Time-Series format, ghi bất đồng bộ vào InfluxDB siêu tốc.

Cầu Nối OPC UA

Đẩy thông số thật vào các Node Tags, cho điểm tựa để Siemens NX MCD dựng phỏng hình 3D Digital Twin.

Điều Phối AI

Chạy Inference AI: Nhận ảnh web gửi lên -> Đẩy vào CNN TFLite. Cấp dữ liệu log -> XGBoost dự báo vụ mùa.

Thời Gian Thực Web

Lập tunnel WebSocket bắn dữ liệu ra Dashboard WebApp < 0.1s. Không cần làm mới trang.

Điều Dưỡng RESTful

Cấp Endpoint JSON bài bản /api/... cho Chart.JS vẽ đồ thị và nút bấm bật tắt Relay từ giao diện GUI.
