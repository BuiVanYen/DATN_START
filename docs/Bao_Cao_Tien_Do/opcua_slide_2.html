FastAPI (Client 1)
  • Chạy trên Raspberry Pi.
  • Kết nối thư viện opcua-asyncio.
  • Nhận dữ liệu từ ESP32 (MQTT) và Ghi / Cập nhật liên tục vào các Node trên Server.
  • Giúp bản sao số 3D biết rõ tình trạng phần cứng hiện tại.

Siemens NX (Client 2)
  • Kết nối trực tiếp qua IP TCP.
  • Đọc dũ liệu: Thấy biến PumpState = True, nó sẽ tạo hiệu ứng nước ảo chảy ra từ ống 3D.
  • Ghi dữ liệu: Khi phao ảo trong MCT bị nước dâng chạm vào, nó sẽ đẩy cờ [Va chạm] = True ngược lại Server để Backend xử lý.
