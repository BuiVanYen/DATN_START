Mô Phỏng Digital Twin 2D

Live Animation
Việc giám sát bằng con số thông thường rất nhàm chán. Do đó, hệ thống tích hợp công nghệ Digital Twin 2D (Bản sao số 2D) bằng SVG/Canvas ngay trên web.

Khi quạt ngoài đời thực quay, cánh quạt trên UI web cũng quay theo. Khi bật bơm, đường ống ảo trên màn hình điện thoại lập tức hiển thị hoạt ảnh dòng nước chảy. Qua đó, chủ vườn nắm bắt chuẩn xác trạng thái thủy động lực học của hệ thống dù ở cách xa ngàn cây số.

Phản hồi từ FastAPI Server

Tận dụng trình duyệt Web Mobile, người chụp trực tiếp lá bằng thẻ <input type="file" capture="camera">. Ảnh đẩy lên Backend chạy MobileNetV2 để phát hiện bệnh lập tức.
