Digital Twin 3D — Siemens NX MCD

Mô Hình 3D (NX CAD)

  • Vẽ 1:1 khung sắt V 2 tầng, 6 máng PVC, bể chứa
  • Đặt chính xác vị trí cảm biến (SHT31, DS18B20, TDS, pH, BH1750, Y26)
  • Đặt vị trí thiết bị chấp hành (bơm, quạt, đèn)
  • Hệ thống ống dẫn nước, van, đường điện

Mô Phỏng Cơ điện tử (NX MCD)

  • Bơm: PWM duty → quay nhanh/chậm theo tốc độ thật
  • Quạt: PWM duty → cánh quạt quay realtime
  • Đèn: ON/OFF → sáng/tắt trên mô hình
  • Mực nước: Y26 → 4 mức hiển thị
  • Sensor: hiển thị giá trị realtime trên 3D

Auto-restart Khi Mất Điện

Raspberry Pi 4 (systemd)

# /etc/systemd/system/hydroponic.service
[Unit]
Description=Hydroponic AIOT System
After=network.target mosquitto.service influxdb.service

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/DATN_YEN/pi4_server
ExecStartPre=/bin/sleep 10
ExecStart=/usr/bin/python3 main.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target

Health Check (cron 5 phút)

#!/bin/bash
for svc in mosquitto influxdb hydroponic; do
    if ! systemctl is-active --quiet $svc; then
        echo "$(date): $svc down, restarting..."
        sudo systemctl restart $svc
    fi
done

ESP32 Failsafe

  • Auto-reconnect WiFi
  • Auto-reconnect MQTT broker
  • Mất MQTT > 5 phút → bơm 15p/tắt 15p tự động
Mini UPS 12V 2A — giữ hệ thống 10-15 phút, đủ để Pi shutdown an toàn
